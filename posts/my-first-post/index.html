我的第一篇Blog

2022年5月10日 建立人生第一個Blog網頁

環境: MAC OS 11.5.2

Path: .文件/website/blog

使用工具介紹

HUGO網頁架構

參考教學: marcolee cfanz

註：cfanz所有用的風格有問題。

選用hugo-theme-bootstrap風格

需安裝

hugo mod npm pack  
npm install  
hugo server

MacDown網頁編輯工具

可用來編輯mackdown格式的文本。 [LINK]

常用指令

啟動 Hugo 伺服器，指令如下。 http://localhost:1313/ 

hugo server -D

新增頁面

hugo new posts/name-of-post/index.zh-tw.md
or
hugo new notes/name-of-note/index.zh-tw.md

用上面的方法會新增一個資料夾(name-of-post)就在***./content/posts裡 所有用到的圖片(ex: img.jpg)都可放在這裡面，要貼圖就下(img.jpg)***即可。

##如何發布到Github上 在blog根目錄下執行

hugo

進到./blog/public裡去更新git

git push -u origin main

Git指令

cd public
git init
git remote add origin https://github.com/crrsn/blog.git
git add .
git ci -m 'first commit'
git push -u origin main
